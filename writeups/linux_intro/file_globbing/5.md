# Multiple globs 

This challenge wants us to match file names by globbing multiple times 

## Solution:

Just cd into the `/challenge/files` directory run the `/challenge/run`  with 
`*p*`
```sh
$ cd /challenge/files
$ /challenge/run *p*
```
